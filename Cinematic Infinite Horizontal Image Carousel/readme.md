# Cinematic Infinite Horizontal Card Carousel

A premium GSAP-powered horizontal card carousel designed to make three visible cards feel like part of an infinite visual stream.

## Concept

This pattern uses:
- three visible cards
- a dominant center focus card
- softened side preview cards
- subtle 3D perspective
- continuous content rotation

The result is a cinematic content switcher that feels more like an editorial motion system than a standard slider.

## Features

- seamless infinite-loop illusion
- horizontal center-focused composition
- smooth GSAP transitions
- subtle 3D rotation for side cards
- blur-based depth hierarchy
- easy dataset swapping
- autoplay plus manual click advance
- premium, restrained motion style
