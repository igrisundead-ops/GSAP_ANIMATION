# Target Dart Result Beat

## Primary Use Case
Use this beat to visualize failure, misalignment, lack of precision, missed performance, or inconsistent outcomes.

## Best Used For
- performance marketing sequences
- “wrong strategy” framing
- misalignment messaging
- analytics storytelling
- precision and targeting metaphors

## Visual Behavior
The target structure enters as a diagrammatic support asset. The dart flies in with directional intent. The copy resolves after the diagram is established.

## Motion Sequence
1. target module enters from the right with blur resolve
2. dart flies into position with directional momentum
3. copy resolves and confirms the message

## Emotional Effect
This beat communicates frustration, misfire, lack of control, and strategic failure while still feeling elegant and premium.

## Works Best With
- target diagrams
- conversion funnels
- KPI visuals
- arrows
- maps
- pointer-based iconography
- result-related language

## Avoid Using When
Avoid for emotional, personal, or trust-driven scenes. This is a symbolic systems/precision beat.

## Customizable Elements
- headline text
- diagram slot
- dart styling
- entry angle
- speed
- scale
- blur intensity

## Retrieval Keywords
target metaphor, missed result beat, dart entry, performance failure visual, diagram support animation

## Agent Notes
This beat pairs well after a portrait authority beat or before a solution reveal. It is a strong modular escalation unit.
